Deskripsi Mata Kuliah

Mata kuliah Ekonometri membekali mahasiswa dengan kemampuan analisis kuantitatif berbasis data menggunakan metode regresi dalam konteks ekonomi dan keuangan. Kajian pertama mencakup regresi dan asumsinya: model regresi linier sederhana dan berganda, berbagai bentuk fungsional, penggunaan variabel dummy, serta diagnosis dan penanganan pelanggaran asumsi klasik OLS. Kajian kedua mencakup model regresi lanjutan: logit/probit, multinomial, ordinal, variabel dependen terbatas (Tobit), model Poisson, analisis deret waktu, serta model ARCH/GARCH untuk volatilitas harga aset.

Capaian Pembelajaran Mata Kuliah (CPMK)
CPMK 1 Mahasiswa mampu menguasai konsep dasar regresi linier, berbagai bentuk fungsional regresi, penggunaan variabel dummy, serta mengidentifikasi dan menangani pelanggaran asumsi klasik (multikolinearitas, heteroskedastisitas, autokorelasi, dan kesalahan spesifikasi model).
CPMK 2 Mahasiswa mampu menganalisis dan menerapkan model-model regresi lanjutan untuk data kategorikal (logit, probit, multinomial, ordinal), data terbatas (Tobit), data cacah (Poisson), serta data deret waktu termasuk uji stasioneritas dan model ARCH/GARCH untuk analisis volatilitas.
