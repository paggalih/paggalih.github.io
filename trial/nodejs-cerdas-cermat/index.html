🚀 Cara Pakai

  1. Install Node.js dari nodejs.org (pilih LTS)
  2. Extract ZIP → buka Terminal/CMD di folder cerdas-cermat-online
  3. Jalankan: npm install
  4. Lalu: npm start
  5. Juri buka http://localhost:3000/juri
  6. Peserta buka http://[IP]:3000/peserta dari HP (WiFi sama)
