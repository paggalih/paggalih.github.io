Sesi 1 — Ketaksamaan Cauchy-Schwarz

60 menit · Teori 15' → Contoh 25' → Latihan 20'

1.1 Bentuk Standar

Cauchy-Schwarz (bentuk jumlah)
Untuk bilangan real $a_1, \ldots, a_n$ dan $b_1, \ldots, b_n$: $$\begin{aligned} & (a_1^2 + a_2^2 + \cdots + a_n^2)(b_1^2 + b_2^2 + \cdots + b_n^2) \\ & \qquad \ge (a_1 b_1 + a_2 b_2 + \cdots + a_n b_n)^2. \end{aligned}$$ Kesamaan iff ada $k$ sehingga $a_i = k\,b_i$ untuk semua $i$ (atau semua $b_i = 0$).

Versi 2-variabel:

$$(a^2 + b^2)(c^2 + d^2) \ge (ac + bd)^2.$$

Bukti elegan (versi 2 variabel): Perhatikan

$$(a^2+b^2)(c^2+d^2) - (ac+bd)^2 = (ad - bc)^2 \ge 0. \quad\blacksquare$$

1.2 Bentuk Engel (Titu's Lemma)

Titu's Lemma — Engel Form
Untuk $a_i \in \mathbb{R}$ dan $b_i > 0$: $$\begin{aligned} & \frac{a_1^2}{b_1} + \frac{a_2^2}{b_2} + \cdots + \frac{a_n^2}{b_n} \\ & \qquad \ge \frac{(a_1 + a_2 + \cdots + a_n)^2}{b_1 + b_2 + \cdots + b_n}. \end{aligned}$$ Kesamaan iff $\dfrac{a_1}{b_1} = \dfrac{a_2}{b_2} = \cdots = \dfrac{a_n}{b_n}$.

Bentuk Engel adalah senjata utama ketaksamaan KSN-K. Hampir semua soal "$\frac{1}{x} + \frac{1}{y} + \frac{1}{z} \ge \ldots$" dapat diserang dengan Titu.

1.3 Kapan Pakai Apa?

Bentuk soalStrategi
Jumlah + hasil kali (semua positif)AM-GM

1.4 Contoh Terbimbing

Contoh 1Aplikasi langsung Cauchy-Schwarz

Soal. Untuk $a, b, c \in \mathbb{R}$, buktikan

$$(a + b + c)^2 \le 3(a^2 + b^2 + c^2).$$

Penyelesaian. C-S dengan $(b_1, b_2, b_3) = (1, 1, 1)$:

$$(a^2 + b^2 + c^2)(1 + 1 + 1) \ge (a + b + c)^2.$$

Maka $3(a^2+b^2+c^2) \ge (a+b+c)^2$. Kesamaan saat $a = b = c$. $\blacksquare$

Contoh 2Titu's Lemma — bentuk pecahan

Soal. Untuk $a, b, c > 0$ buktikan

$$\frac{a^2}{b} + \frac{b^2}{c} + \frac{c^2}{a} \ge a + b + c.$$

Penyelesaian. Titu langsung:

$$\begin{aligned} \frac{a^2}{b} + \frac{b^2}{c} + \frac{c^2}{a} &\ge \frac{(a+b+c)^2}{a+b+c} \\ &= a+b+c. \quad\blacksquare \end{aligned}$$

Kesamaan iff $\dfrac{a}{b} = \dfrac{b}{c} = \dfrac{c}{a}$, yaitu $a = b = c$.

Contoh 3Nesbitt — soal klasik

Soal. Untuk $a, b, c > 0$ buktikan

$$\frac{a}{b+c} + \frac{b}{c+a} + \frac{c}{a+b} \ge \frac{3}{2}.$$

Penyelesaian (Titu). Tulis ulang:

$$\begin{aligned} & \frac{a^2}{a(b+c)} + \frac{b^2}{b(c+a)} + \frac{c^2}{c(a+b)} \\ & \qquad \ge \frac{(a+b+c)^2}{a(b+c) + b(c+a) + c(a+b)}. \end{aligned}$$

Penyebut $= 2(ab + bc + ca)$. Maka

$$\text{LHS} \ge \frac{(a+b+c)^2}{2(ab+bc+ca)}.$$

Kita tahu $(a+b+c)^2 \ge 3(ab+bc+ca)$ (dari Contoh 1), jadi

$$\text{LHS} \ge \frac{3(ab+bc+ca)}{2(ab+bc+ca)} = \frac{3}{2}. \quad\blacksquare$$
💡 Trik penting. Untuk pecahan $\dfrac{a}{X}$, kalikan atas-bawah dengan $a$ jadi $\dfrac{a^2}{aX}$ — bentuk baku Titu.

1.5 Latihan Mandiri

  • Buktikan untuk $x, y, z \in \mathbb{R}$: $(x+y+z)^2 \le 3(x^2+y^2+z^2)$ langsung dari SOS (tanpa C-S).
    Kunci
    $3(x^2+y^2+z^2) - (x+y+z)^2 = 2(x^2+y^2+z^2) - 2(xy+yz+zx) = (x-y)^2 + (y-z)^2 + (z-x)^2 \ge 0$. $\blacksquare$
  • $a, b, c > 0$ dan $a + b + c = 1$. Buktikan $\dfrac{1}{a} + \dfrac{1}{b} + \dfrac{1}{c} \ge 9$.
    Kunci
    Titu: $\dfrac{1^2}{a} + \dfrac{1^2}{b} + \dfrac{1^2}{c} \ge \dfrac{(1+1+1)^2}{a+b+c} = \dfrac{9}{1} = 9$. $\blacksquare$
  • $a, b, c > 0$. Buktikan $\dfrac{a^2}{b+c} + \dfrac{b^2}{c+a} + \dfrac{c^2}{a+b} \ge \dfrac{a+b+c}{2}$.
    Kunci
    Titu: $\text{LHS} \ge \dfrac{(a+b+c)^2}{(b+c)+(c+a)+(a+b)} = \dfrac{(a+b+c)^2}{2(a+b+c)} = \dfrac{a+b+c}{2}$. $\blacksquare$
  • Carilah maksimum $\sqrt{x} + \sqrt{2-x}$ untuk $0 \le x \le 2$.
    Kunci
    C-S: $(\sqrt{x}\cdot 1 + \sqrt{2-x}\cdot 1)^2 \le (x + (2-x))(1+1) = 4$. Jadi $\sqrt{x} + \sqrt{2-x} \le 2$, maksimum $= \mathbf{2}$ saat $x = 1$.
  • $x_1, \ldots, x_n > 0$ dengan $x_1 + \cdots + x_n = 1$. Buktikan $\dfrac{x_1^2}{1+x_1} + \cdots + \dfrac{x_n^2}{1+x_n} \ge \dfrac{1}{n+1}$.
    Kunci
    Titu: LHS $\ge \dfrac{(x_1 + \cdots + x_n)^2}{(1+x_1) + \cdots + (1+x_n)} = \dfrac{1}{n+1}$. $\blacksquare$

Sesi 2 — Persamaan Fungsional Sederhana

60 menit · Konsep 15' → Contoh 25' → Latihan 20'

2.1 Apa itu Persamaan Fungsional?

Persamaan yang harus dipenuhi fungsi $f$ untuk semua nilai input — bukan persamaan dengan variabel $x$. Kita harus mencari semua $f$ yang memenuhi.

Contoh sederhana: "Cari semua $f: \mathbb{R} \to \mathbb{R}$ dengan $f(x+y) = f(x) + f(y)$ untuk semua $x, y$."

2.2 Strategi Substitusi Standar

SubstitusiYang biasanya didapat
$x = y = 0$Nilai $f(0)$
$x = 0$ atau $y = 0$Hubungan dengan $f(0)$
$y = x$Bentuk kuadrat: $f(2x)$ atau $f(x^2)$
$y = -x$Apakah $f$ ganjil/genap?
$x = y = 1$Nilai $f(1), f(2)$
Tukar $x \leftrightarrow y$Simetri: ekstrak info baru
💡 Aturan emas. Mulai selalu dengan substitusi paling sederhana ($x=y=0$). Kumpulkan hasil-hasil parsial. Tebak bentuk $f$, lalu buktikan tebakan benar.

2.3 Contoh Terbimbing

Contoh 4Cauchy aditif

Soal. Cari semua $f: \mathbb{Q} \to \mathbb{Q}$ yang memenuhi $f(x+y) = f(x) + f(y)$ untuk semua $x, y \in \mathbb{Q}$.

Penyelesaian.

Langkah 1. $x = y = 0$: $f(0) = 2f(0) \Rightarrow f(0) = 0$.

Langkah 2. Misal $f(1) = c$. Induksi pada bilangan asli:

$$f(n) = f(\underbrace{1+1+\cdots+1}_{n}) = nc, \quad n \in \mathbb{N}.$$

Langkah 3. $y = -x$: $f(0) = f(x) + f(-x) \Rightarrow f(-x) = -f(x)$. Jadi $f(n) = nc$ untuk $n \in \mathbb{Z}$.

Langkah 4. Untuk $\dfrac{p}{q} \in \mathbb{Q}$: $q \cdot f\!\left(\dfrac{p}{q}\right) = f(p) = pc \Rightarrow f\!\left(\dfrac{p}{q}\right) = \dfrac{p}{q} \cdot c$.

Jawaban: $f(x) = cx$ untuk konstanta $c \in \mathbb{Q}$ apa saja.

Contoh 5Substitusi pintar

Soal. Cari semua $f: \mathbb{R} \to \mathbb{R}$ yang memenuhi $f(x) + 2f(1-x) = x^2$ untuk semua $x$.

Penyelesaian. Substitusi $x \to 1-x$:

$$f(1-x) + 2f(x) = (1-x)^2.$$

Kita punya sistem dua persamaan:

$$\begin{cases} f(x) + 2f(1-x) = x^2 \\ 2f(x) + f(1-x) = (1-x)^2 \end{cases}$$

Eliminasi: kali persamaan kedua dengan 2, kurangi dari pertama:

$$\begin{aligned} -3f(x) &= x^2 - 2(1-x)^2 \\ f(x) &= \frac{2(1-x)^2 - x^2}{3} = \frac{x^2 - 4x + 2}{3}. \end{aligned}$$

Cek: Substitusi balik mengkonfirmasi hasil. ✓

Contoh 6Tebak & buktikan

Soal. Cari semua $f: \mathbb{R} \to \mathbb{R}$ dengan $f(xy) = f(x)\,f(y)$ untuk semua $x, y$ dan $f$ kontinu.

Penyelesaian.

Langkah 1. $x = y = 0$: $f(0) = f(0)^2 \Rightarrow f(0) \in \{0, 1\}$.

Langkah 2. Jika $f(0) = 1$: ambil $y = 0$ saja: $f(0) = f(x)f(0) = f(x)$ untuk semua $x$. Jadi $f \equiv 1$.

Langkah 3. Jika $f(0) = 0$: cek apakah ada $f$ tak-nol. Misal $f(1) = a$. Dari $f(1\cdot 1) = a^2 \Rightarrow a = a^2 \Rightarrow a \in \{0,1\}$.

Kalau $a = 0$: $f(x) = f(x \cdot 1) = f(x) \cdot 0 = 0$ untuk semua $x$. Jadi $f \equiv 0$.

Kalau $a = 1$: tebak $f(x) = x^k$ atau $f(x) = |x|^k$. Cek di soal: $(xy)^k = x^k y^k$ ✓ — bekerja. Dengan kontinuitas, jawaban umum: $f(x) = |x|^k$ atau $f(x) = |x|^k \cdot \text{sgn}(x)^m$ untuk $k \ge 0$.

Jawaban (umum): $f \equiv 0$, $f \equiv 1$, atau $f(x) = |x|^k$ (dan variannya) untuk $k > 0$.

(Untuk KSN-K, cukup tunjukkan ide tebakan; bukti lengkap kontinuitas tidak wajib.)

2.4 Latihan Mandiri

  • Cari semua $f: \mathbb{R} \to \mathbb{R}$ dengan $f(x) - f(-x) = 2x$ untuk semua $x$.
    Kunci
    Substitusi $x \to -x$: $f(-x) - f(x) = -2x$, sama dengan persamaan asli (konsisten). Tulis $f(x) = g(x) + h(x)$ dengan $g$ genap, $h$ ganjil. Persamaan jadi $2h(x) = 2x \Rightarrow h(x) = x$. Bagian genap $g$ bebas. Jawaban: $f(x) = x + g(x)$ dengan $g$ fungsi genap apa saja.
  • $f: \mathbb{R} \to \mathbb{R}$ memenuhi $f(x) + f\!\left(\dfrac{1}{x}\right) = x$ untuk semua $x \neq 0$. Tentukan $f$.
    Kunci
    Substitusi $x \to 1/x$: $f(1/x) + f(x) = 1/x$. Bandingkan dengan persamaan asli: $x = 1/x$ untuk semua $x$ — kontradiksi kecuali $x = \pm 1$. Jawaban: tidak ada $f$ yang memenuhi. (Soal jebakan — penting untuk dilatih!)
  • $f: \mathbb{R} \to \mathbb{R}$ memenuhi $2f(x) + f(1-x) = x^2$. Tentukan $f(x)$.
    Kunci
    Sub $x \to 1-x$: $2f(1-x) + f(x) = (1-x)^2$. Kali persamaan asli dengan 2, kurangi: $3f(x) = 2x^2 - (1-x)^2 = x^2 + 2x - 1$. Jadi $\mathbf{f(x) = \dfrac{x^2 + 2x - 1}{3}}$.
  • $f: \mathbb{N} \to \mathbb{N}$ memenuhi $f(1) = 1$ dan $f(n+1) = f(n) + 2n + 1$ untuk semua $n$. Tentukan rumus $f(n)$.
    Kunci
    $f(2) = 1+3 = 4$, $f(3) = 4+5 = 9$, $f(4) = 9+7 = 16$. Tebak $\mathbf{f(n) = n^2}$. Bukti induksi: $f(n+1) = n^2 + 2n + 1 = (n+1)^2$. ✓
  • Cari semua $f: \mathbb{R} \to \mathbb{R}$ dengan $f(x+y) = f(x) + f(y) + xy$ untuk semua $x, y \in \mathbb{R}$ dan $f(1) = 1$.
    Kunci
    Misal $g(x) = f(x) - \dfrac{x^2}{2}$. Cek: $g(x+y) = f(x+y) - \dfrac{(x+y)^2}{2} = f(x) + f(y) + xy - \dfrac{x^2 + 2xy + y^2}{2} = g(x) + g(y)$. Jadi $g$ aditif Cauchy. Untuk $\mathbb{Q}$: $g(x) = cx$, dengan $g(1) = f(1) - \tfrac{1}{2} = \tfrac{1}{2}$. Maka $\mathbf{f(x) = \dfrac{x^2}{2} + \dfrac{x}{2} = \dfrac{x(x+1)}{2}}$.

Sesi 3 — Barisan, Deret & Teleskop

40 menit · Konsep 12' → Contoh 15' → Latihan 13'

3.1 Tiga Teknik Inti

1. Teleskop
Tulis ulang $a_k = b_k - b_{k-1}$ (selisih). Maka $$\sum_{k=1}^{n} a_k = b_n - b_0.$$
2. Pecahan Parsial
Untuk pecahan rasional, pecah jadi suku-suku yang lebih sederhana. Pola umum: $$\frac{1}{k(k+1)} = \frac{1}{k} - \frac{1}{k+1},$$ $$\frac{1}{k(k+1)(k+2)} = \frac{1}{2}\left[\frac{1}{k(k+1)} - \frac{1}{(k+1)(k+2)}\right].$$
3. Rekursi Linear
Untuk $a_{n+1} = pa_n + qa_{n-1}$, akar persamaan karakteristik $x^2 - px - q = 0$ memberi solusi umum $$a_n = A\,r_1^n + B\,r_2^n.$$

3.2 Rumus Penjumlahan Penting

DeretRumus
$1 + 2 + \cdots + n$$\dfrac{n(n+1)}{2}$
$1^2 + 2^2 + \cdots + n^2$$\dfrac{n(n+1)(2n+1)}{6}$
$1^3 + 2^3 + \cdots + n^3$$\left[\dfrac{n(n+1)}{2}\right]^2$
Geometri $a + ar + \cdots + ar^{n-1}$$\dfrac{a(1-r^n)}{1-r}$ ($r \neq 1$)
$\dfrac{1}{1\cdot 2} + \dfrac{1}{2\cdot 3} + \cdots + \dfrac{1}{n(n+1)}$$1 - \dfrac{1}{n+1} = \dfrac{n}{n+1}$

3.3 Contoh Terbimbing

Contoh 7Teleskop dengan akar

Soal. Hitung $\displaystyle\sum_{k=1}^{99} \frac{1}{\sqrt{k} + \sqrt{k+1}}$.

Penyelesaian. Rasionalkan tiap suku:

$$\begin{aligned} \frac{1}{\sqrt{k} + \sqrt{k+1}} &\cdot \frac{\sqrt{k+1} - \sqrt{k}}{\sqrt{k+1} - \sqrt{k}} \\ &= \sqrt{k+1} - \sqrt{k}. \end{aligned}$$

Teleskop:

$$\sum_{k=1}^{99} (\sqrt{k+1} - \sqrt{k}) = \sqrt{100} - \sqrt{1} = \mathbf{9}.$$
Contoh 8Rekursi Fibonacci-like

Soal. $a_1 = 1, a_2 = 3$, $a_{n+1} = 5a_n - 6a_{n-1}$. Cari rumus tertutup $a_n$.

Penyelesaian. Persamaan karakteristik: $x^2 - 5x + 6 = 0 \Rightarrow (x-2)(x-3) = 0$, akar $r_1 = 2, r_2 = 3$.

Solusi umum: $a_n = A\cdot 2^n + B\cdot 3^n$. Dari syarat awal:

$$\begin{cases} 2A + 3B = 1 \\ 4A + 9B = 3 \end{cases} \;\Longrightarrow\; A = 0, \ B = \tfrac{1}{3}.$$

Maka $\mathbf{a_n = \dfrac{3^n}{3} = 3^{n-1}}$. Cek: $a_1 = 1, a_2 = 3, a_3 = 9$. ✓

Contoh 9Pecahan parsial

Soal. Hitung $\displaystyle S = \sum_{k=1}^{n} \frac{1}{k(k+1)(k+2)}$.

Penyelesaian. Gunakan dekomposisi

$$\frac{1}{k(k+1)(k+2)} = \frac{1}{2}\left[\frac{1}{k(k+1)} - \frac{1}{(k+1)(k+2)}\right].$$

Teleskop:

$$\begin{aligned} S &= \frac{1}{2}\left[\frac{1}{1\cdot 2} - \frac{1}{(n+1)(n+2)}\right] \\ &= \frac{1}{4} - \frac{1}{2(n+1)(n+2)}. \end{aligned}$$

3.4 Latihan Mandiri

  • Hitung $\displaystyle\sum_{k=1}^{n} k(k+1)$.
    Kunci
    $k(k+1) = k^2 + k$. Jadi $\sum = \dfrac{n(n+1)(2n+1)}{6} + \dfrac{n(n+1)}{2} = \dfrac{n(n+1)}{6}\left[(2n+1) + 3\right] = \mathbf{\dfrac{n(n+1)(n+2)}{3}}$.
  • Hitung $\displaystyle\sum_{k=1}^{50} \frac{1}{k^2 + k}$.
    Kunci
    $\dfrac{1}{k^2+k} = \dfrac{1}{k(k+1)} = \dfrac{1}{k} - \dfrac{1}{k+1}$. Teleskop: $1 - \dfrac{1}{51} = \mathbf{\dfrac{50}{51}}$.
  • $a_1 = 2$, $a_{n+1} = 3a_n - 2$. Cari rumus tertutup $a_n$.
    Kunci
    Cari titik tetap: $L = 3L - 2 \Rightarrow L = 1$. Misal $b_n = a_n - 1$, maka $b_{n+1} = 3b_n$ dan $b_1 = 1$. Jadi $b_n = 3^{n-1}$, sehingga $\mathbf{a_n = 3^{n-1} + 1}$.
  • Hitung $\displaystyle\sum_{k=1}^{n} k \cdot k!$.
    Kunci
    $k \cdot k! = (k+1)! - k!$. Teleskop: $\sum = (n+1)! - 1! = \mathbf{(n+1)! - 1}$.
  • Barisan Fibonacci $F_1 = F_2 = 1$, $F_{n+1} = F_n + F_{n-1}$. Buktikan $F_1 + F_2 + \cdots + F_n = F_{n+2} - 1$.
    Kunci
    Tulis $F_k = F_{k+2} - F_{k+1}$. Maka $\sum_{k=1}^n F_k = \sum_{k=1}^n (F_{k+2} - F_{k+1})$ — teleskop $= F_{n+2} - F_2 = F_{n+2} - 1$. $\blacksquare$

Latihan Gabungan — Mini Olimpiade

20 menit · Pilih 3 dari 5 · Materi pertemuan 1 + 2
  • $a, b, c > 0$ dan $abc = 1$. Buktikan $\dfrac{1}{a^2(b+c)} + \dfrac{1}{b^2(c+a)} + \dfrac{1}{c^2(a+b)} \ge \dfrac{3}{2}$.
    Kunci
    Karena $abc = 1$: $\dfrac{1}{a^2(b+c)} = \dfrac{bc}{a(b+c) \cdot abc} = \dfrac{bc/a}{b+c}$. Substitusi $x = 1/a, y=1/b, z=1/c$, syarat $xyz = 1$: $\sum \dfrac{x^2}{y+z}$. Titu: $\ge \dfrac{(x+y+z)^2}{2(x+y+z)} = \dfrac{x+y+z}{2}$. Dari AM-GM $x+y+z \ge 3\sqrt[3]{xyz}=3$. Jadi $\ge \dfrac{3}{2}$. $\blacksquare$
  • Cari semua $f: \mathbb{R} \to \mathbb{R}$ dengan $f(x+y) + f(x-y) = 2f(x)f(y)$ dan $f$ tidak nol.
    Kunci
    $x = y = 0$: $2f(0) = 2f(0)^2 \Rightarrow f(0) \in \{0, 1\}$. Jika $f(0) = 0$, $y = 0$: $2f(x) = 0$ untuk semua $x$ — kontradiksi. Jadi $f(0) = 1$. $x = 0$: $f(y) + f(-y) = 2f(y) \Rightarrow f$ genap. Tebakan: $f(x) = \cos(kx)$ atau $f(x) = \cosh(kx)$. Verifikasi cocok dengan rumus jumlah-selisih cosinus. Jawaban: $f(x) = \cos(kx)$ atau $\cosh(kx)$ untuk $k \in \mathbb{R}$ (juga $f \equiv 1$ sebagai kasus $k=0$).
  • Hitung $\displaystyle\sum_{k=1}^{100} \frac{1}{k(k+1)(k+2)(k+3)}$.
    Kunci
    Pola umum: $\dfrac{1}{k(k+1)(k+2)(k+3)} = \dfrac{1}{3}\left[\dfrac{1}{k(k+1)(k+2)} - \dfrac{1}{(k+1)(k+2)(k+3)}\right]$. Teleskop: $\dfrac{1}{3}\left[\dfrac{1}{1\cdot 2 \cdot 3} - \dfrac{1}{101 \cdot 102 \cdot 103}\right] = \dfrac{1}{18} - \dfrac{1}{3 \cdot 101 \cdot 102 \cdot 103}$.
  • $a, b, c$ adalah akar $x^3 - 6x^2 + 11x - 6 = 0$. Hitung $\dfrac{1}{a^2} + \dfrac{1}{b^2} + \dfrac{1}{c^2}$.
    Kunci
    Vieta: $a+b+c = 6, ab+bc+ca = 11, abc = 6$. $\dfrac{1}{a^2}+\dfrac{1}{b^2}+\dfrac{1}{c^2} = \dfrac{(ab+bc+ca)^2 - 2abc(a+b+c)}{(abc)^2} = \dfrac{121 - 72}{36} = \mathbf{\dfrac{49}{36}}$.
  • Untuk $a, b, c > 0$ buktikan $\dfrac{a^3}{b+c} + \dfrac{b^3}{c+a} + \dfrac{c^3}{a+b} \ge \dfrac{a^2+b^2+c^2}{2}$.
    Kunci
    Tulis $\dfrac{a^3}{b+c} = \dfrac{(a^2)^2}{a(b+c)}$. Titu: $\sum \dfrac{(a^2)^2}{a(b+c)} \ge \dfrac{(a^2+b^2+c^2)^2}{a(b+c)+b(c+a)+c(a+b)} = \dfrac{(a^2+b^2+c^2)^2}{2(ab+bc+ca)}$. Karena $ab+bc+ca \le a^2+b^2+c^2$ (dari SOS), maka $\dfrac{(a^2+b^2+c^2)^2}{2(ab+bc+ca)} \ge \dfrac{a^2+b^2+c^2}{2}$. $\blacksquare$

Refleksi setelah pertemuan 1 + persiapan pertemuan 2

Refleksi dari Pertemuan 1

  • Cek PR Pertemuan 1 di awal — minta 1-2 siswa menjelaskan jawaban di papan.
  • Identifikasi siswa yang kesulitan dengan identitas dasar. Mereka butuh latihan ekstra di luar kelas.
  • Catat materi mana yang lambat — mungkin perlu diulang sebelum Sesi 1 hari ini.

Selama Mengajar Pertemuan 2

  • Cauchy-Schwarz adalah bintang utama. Pastikan semua siswa hafal Titu's Lemma — ini akan dipakai sangat sering.
  • Untuk persamaan fungsional, jangan paksa siswa menemukan jawaban sendiri pertama kali. Tunjukkan pola substitusi, lalu mereka tiru.
  • Sesi 3 paling padat — siapkan timer. Jangan tergoda menjelaskan terlalu detail rumus deret; siswa cukup tahu cara teleskop.
  • Bila siswa bingung memilih strategi (AM-GM vs C-S vs SOS), kembali ke tabel di bagian 1.3.

Untuk Pertemuan 3

  • Kombinatorika dasar: prinsip pencacahan, kombinasi, prinsip lubang merpati.
  • Atau: Geometri olimpiade — kesebangunan, garis berat, lingkaran luar/dalam segitiga.
  • Atau: Teori bilangan — keterbagian, modular, FPB & KPK.
  • Pilih sesuai keseimbangan — jangan tinggal di aljabar terus-menerus.
Pesan penutup. Pertemuan kedua adalah lompatan. Siswa yang bertahan dari pertemuan ini biasanya akan terus jalan. Beri pengakuan untuk usaha mereka, bukan hanya untuk jawaban yang benar.
