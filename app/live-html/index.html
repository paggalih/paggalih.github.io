Berikut adalah perbaikan dan penjelasannya. ### 1\. Masalah Fullscreen Otomatis (Default) Penyebabnya adalah ada fungsi JavaScript `checkScreenSize()` yang langsung dijalankan saat halaman dimuat. Fungsi ini memeriksa lebar layar, dan jika lebih dari 1200px, ia akan otomatis mengaktifkan mode fullscreen. **Solusi:** Hapus atau jadikan komentar pada pemanggilan fungsi tersebut saat halaman pertama kali dimuat. ```javascript // // Jalankan saat pertama kali dimuat // checkScreenSize(); // <-- BARIS INI DINONAKTIFKAN // // Jalankan juga saat ukuran layar berubah // window.addEventListener('resize', checkScreenSize); // <-- BARIS INI JUGA BISA DINONAKTIFKAN JIKA TIDAK PERLU ``` ----- ### 2\.
