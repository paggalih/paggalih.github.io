Huashan Swords Algorithm

Multi-Objective VRP Solver for Heterogeneous Fleet with Time Windows

HSSA (Zhang et al., 2026) solves multi-objective Vehicle Routing Problems with time windows using heterogeneous fleets (owned/rented, EV/CV). Inspired by the ancient Chinese Huashan Martial Arts Convention, it uses hierarchical search with regional leaders, skill transfer, and peer competition to optimize distribution cost, CO₂ emissions, and vehicle count simultaneously.
